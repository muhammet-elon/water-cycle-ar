Android note

Android browsers typically need a .glb (or .gltf) file to launch AR (Scene Viewer / WebXR). This page currently hosts only a .usdz, so Android will likely download/open the file instead of AR.

If you add a Water_Cycle.glb, you can upgrade this page to support Android too.
